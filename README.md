# tundoku App 📚

**「積読（つんどく）」を可視化して、読書をもっと楽しく。**

Tundoku Appは、手元にある未読本をシンプルに管理し、読書の進捗をサポートするためのWebアプリケーションです。
面倒な設定抜きで、誰でも直感的に自分の「本棚」を整理できます。

---

## 🌟 特徴
- **即座に利用可能**: 現在、ログイン不要で誰でもすぐに積読リストを作成できます。
- **直感的なステータス管理**: 「積読」「読書中」「読了」の状態をワンクリックで切り替え。
- **ミニマルなUI/UX**: 読書の邪魔をしない、シンプルで清潔感のあるデザイン。
- **モバイル対応**: 外出先の書店やカフェでも、スマホからサッと自分の在庫を確認。

## 🚀 使い方
1. **アクセス**: [Tundoku App](https://tundoku-app.onrender.com/) へアクセス。
2. **本を追加**: 「本を追加」ボタンから、手元にある積読本を登録。
3. **読書開始**: 読み始めたらステータスを更新して、読書欲をキープ。
4. **読了**: 読み終えたらリストを整理して、次の1冊へ！
